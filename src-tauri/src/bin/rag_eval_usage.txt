rag_eval · RAG 评估 CLI

跑一组 (query, expected) fixture，输出 recall@K 和 MRR。

USAGE:
  rag_eval --workspace <PATH> --fixture <JSON>
           [--k N] [--provider P] [--model M] [--dim D]
           [--base-url URL] [--api-key KEY]

FLAGS:
  --workspace, -w   已索引过的 markio workspace 绝对路径
  --fixture, -f     fixture JSON 路径
  --k               recall@K 的 K，默认 10
  --provider        ollama / openai / 自定义；默认 ollama
  --model           embedding 模型；默认 nomic-embed-text
  --dim             embedding 维度；默认 768
  --base-url        provider base URL（ollama 默认 http://localhost:11434）
  --api-key         provider API key（openai 时必填）
  --help, -h        显示本帮助

FIXTURE JSON 格式:
  { "fixtures": [ { "query": "...", "expected": ["docs/x.md", ...] } ] }

注意：本 CLI 不重建索引；workspace 必须事先在 markio 设置里完成索引。
