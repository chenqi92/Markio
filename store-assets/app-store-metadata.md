Markio is a local-first Markdown reader and writing app built for long-form notes, knowledge work, and technical documentation.

- Optional AI: Connect OpenAI, Anthropic, Gemini, DeepSeek, Ollama, or a custom compatible endpoint. Credentials are stored in the OS keychain.
- Export and publishing: Export to PDF or HTML, copy Markdown/HTML snippets, and preview WeChat-ready formatting.

Markio does not upload your notes by default and does not run ads, tracking, or analytics. Network features such as AI, sync, and image upload only run when you configure them.

## App Review Notes

Markio is a local-first Markdown desktop app. No account is required.

To review the core flow:

1. Launch Markio.
2. Choose or create a local folder with Markdown files.
3. Open or create a .md file.
4. Try source, split, WYSIWYG, and preview modes.
5. Use command palette, full-text search, export, and settings.

Optional AI, cloud sync, image upload, Apple Notes import, and WeChat workflows require user-provided credentials or explicit user action. These are not required to use the app.

Network entitlement is used for optional AI providers, sync/storage providers, update checks, OAuth flows, and image upload. Apple Events automation is used only for user-invoked Apple Notes import and macOS share workflows.

Export compliance draft:

The app uses standard encryption only for HTTPS/TLS network connections and operating-system credential storage. It does not implement proprietary cryptography.

Privacy answer draft:

No tracking. No ads. No analytics. The developer does not operate a Markio account or backend. If App Store Connect requires data collection disclosure for optional third-party integrations, use "Other User Content" for "App Functionality", not linked to the user, not used for tracking.
